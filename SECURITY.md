# Security Policy

## Versions supportees

La version maintenue est la derniere version presente sur la branche `main`.

## Signaler un probleme

Pour un probleme de securite ou un comportement exploitable en jeu, ouvrir une issue privee si disponible ou contacter directement le mainteneur du depot.

Inclure autant que possible:

- version du serveur;
- version Java;
- version du plugin;
- configuration concernee;
- etapes de reproduction;
- logs utiles.
